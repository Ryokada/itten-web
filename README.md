# 一天web

## 概要

草野球チーム一天のWebシステムのコードです。



## スタック

### フロント

* Nextjs 13 (AppRouter)
  * React 18

* Tailewind CSS 3
* TypeScript

### バックエンド/インフラ

* Firebase
* Vercel
* LINE Messaging API



## 開発

開発は `./itten-web-front`フォルダで行います。

```
npm i
npm run dev
```

`http://localhost:3000/`でアクセスできます。
